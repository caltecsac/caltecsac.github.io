Area de IP Networking

Caltec es representante y socio estratégico de diversas marcas de reconocimiento mundial que ayudan a proveer equipamiento para implementación de soluciones de IP Networking en el mercado peruano. A través de sus representadas, Caltec brinda soluciones llave en mano para proyectos de diferente envergadura entre los que destacan:

  • Soluciones de acceso de banda ancha bajo tecnología xDSL (ADSL, VDSL, G.SHDSL). Soluciones WiFi para ambientes Indoor y Outdoor.

  • Soluciones de Seguridad para Redes LAN con Firewall para soportar hasta 500 PCs conectadas.

  • Soluciones de Interconexión de redes LAN usando Servidores y concentradores de IPSec VPN.

  • Sistema de optimización de redes LAN usando Filtro de Contenidos, bloqueo de páginas web, monitoreo del tráfico interno.

  • Solución para sistema de radio enlace para soluciones punto a punto y punto multipunto en banda libres no licenciadas (2.4GHz y 5.8GHz) y licenciadas que permiten cubrir hasta 40Km.

  • Solución de equipamiento 3G/4G para acceso a Internet para zonas donde no hay facilidades técnicas para colocar un sistema cableado, pero sí cuenta con cobertura celular de datos.
